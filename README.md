# CarpetBotLimit

A very simple server-side Fabric mod that intercepts Carpet's `/player` command to limit fake player creation to `bot_<playerName>` per player.

## What it does (For non-OP players)

* **Spawn**: You can only spawn a bot named `bot_<your_name>`.
* **Shadow**: You can only target yourself.
* **Control**: You can only control yourself or your own bot.

## Requirements

* **Server-side only**
* Requires **Carpet Mod**
